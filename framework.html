The Framework

The impossible pentagon, six compromises, and a post-subsidy architecture for Bitcoin.

The Impossible Pentagon

Bitcoin needs five properties to fulfill its promise. Today, the subsidy pays for all five simultaneously. Post-subsidy, they compete. "Pick any three. You can't have all five."

Ten Tensions, Five Vertices

Every pair of vertices creates a tension. You can't maximize both sides. The subsidy masked all five tensions simultaneously. As it disappears, every tension activates at once.

The Equilibrium: Post-Subsidy Architecture

The impossible pentagon means accepting tradeoffs. Here's where to set each dial, and what it costs.

Six Compromises

COMPROMISE 1

Accept That L1 Is Not For Daily Coffee

Give up: Full accessibility for small transactions on L1.
Gain: Block space preserved for high-value settlement, sustainable fee rates.

L2 must remain non-custodial or you've just rebuilt banks.

COMPROMISE 2

Move Data Off L1 by Making L3 Better

Give up: The idea that you can legislate inscriptions away.
Gain: Data moves off-chain because L3 is cheaper, faster, and more capable. No fork needed.

36-byte anchors on L1, full data on L3. Win on economics, not politics.

COMPROMISE 3

Accept That Inscriptions Die From Irrelevance, Not Restriction

Give up: The moral crusade against inscriptions.
Gain: A migration path that data users actually want to take.

L3 offers larger files, updateable metadata, search, collections — things L1 can never do.

COMPROMISE 4

Build a Node Incentive (The Missing Business Model)

Give up: The fiction that nodes run on altruism.
Gain: Sustainable node operation, maintained decentralization.

Pragmatic order: reduce costs (Utreexo) → formalize market (relay services) → protocol payment (last resort).

COMPROMISE 5

Design for AI Coexistence, Not Exclusion

Give up: The idea that Bitcoin is humans-only.
Gain: A fee market that works when machines show up.

Reserved block space for simple transfers. Progressive fee curves. Time-weighted priority.

COMPROMISE 6

Accept the Sovereignty Ceiling

Give up: Belief that Bitcoin resists all nation states simultaneously.
Gain: Realistic defense design: resilience over invulnerability.

Target: resist any single nation. Mining across 10+ jurisdictions. No pool > 25%.

The Post-Subsidy Architecture

Three-Layer DesignArchitecture
Layer 1: Settlement + Anchor
├── Sound money transactions + 36-byte data anchors
├── Revenue: value fees + anchor fees + AI premium + L2 fees
├── No consensus changes required
└── Target: $3-5B/year security budget

Layer 2: Human Transaction Layer
├── Lightning Network (payments)
├── Ark / other non-custodial protocols
├── Critical: NON-CUSTODIAL exit to L1 must always exist
└── This is where humans live day-to-day

Layer 3: Data + Application Layer
├── Anchoring protocol (commit hash on L1, store data off-chain)
├── AI agent coordination layer
├── Inscription migration target
└── Bitcoin's security without Bitcoin's block space

Implementation Sequence

PHASE 1: Now – 2026 • Build

Ship L3

  • L3 working code: anchoring protocol, P2P network, CLI tools
  • Node optimization contributions (pruning, Utreexo research)
  • Education: Pentagon framework, post-subsidy modeling
  • Fee market research: Model the numbers. Verify or falsify.
PHASE 2: 2026 – 2028 • Grow

Win on Economics

  • Make L3 10x cheaper and better than L1 inscriptions
  • Creator tools, galleries, search/discovery on L3
  • Batch anchoring (Merkle trees, many docs per tx)
  • Adoption: make "anchored to Bitcoin" the standard
PHASE 3: 2028 – 2032 • Scale

Deploy at Scale

  • Subsidy drops to 0.78 BTC/block. Pressure becomes undeniable.
  • L3 anchoring at scale (migration path for data users)
  • Anchor fees become meaningful part of miner revenue
  • Inscriptions decline from economic irrelevance, not restriction
PHASE 4: 2032 – 2036 • The Test

Subsidy ~0. Does It Work?

  • Does the fee market sustain $3B+/year?
  • Is mining still distributed across 10+ jurisdictions?
  • Can humans still access L1 for settlement?
  • If yes: Bitcoin survives as sound money for humans.
  • If no: back to the pentagon. Adjust the dials.
There is no solution where everyone gets everything. There is a solution where everyone gets enough. The difference between those two sentences is the difference between ideology and engineering. Bitcoin was built by an engineer. It should be saved by engineering, not by winning an argument.
